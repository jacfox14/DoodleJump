This program allows the user to play our version popular mobile game doodle jump on their desktop, we call it Andy Jump! 

To navigate the menus, use the up and down arrow keys to select text. Use Enter/Return to confirm selection. For the test cases menu,
click on the sprite in the lower right corner from the main menu.

To play the game, press the "A" key to go left and the "D" key to go to the right. press the space bar to shoot a bullet
